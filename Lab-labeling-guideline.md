# Ethernet cabling
## Labeling practices
- Heat shrink labels on each end with the port number and switch number associated with the run as well as where it goes, on both ends.
## Color coding
- Orange = Backbone/uplink
- Black = Servers/Equipment
- Yellow = WIFI/Poe
- Blue = Network data
- Green = CCTV/Security
- White = Telephony/Voice
- Purple = Storage
